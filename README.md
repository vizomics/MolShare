# The MolShare GitHub repository (VIZomics/MolShare) 

serves as the community-driven fork of NCBI's iCn3D, evolving it into a collaborative, extensible open-source platform for web-based 3D biomolecular visualization and analysis.
Core Role in Community Development

Foundation for collective evolution: It preserves iCn3D's core strengths — interactive 3D viewing, synchronized 1D/2D/3D representations, structural analysis (contacts, symmetry, mutations, electrostatics), batch processing (Node.js/Python scripts), and FAIR sharing via permanent URLs encoding full scenes, data, annotations, and analysis history — while enabling unrestricted community-led extensions.

Hub for open contributions: Developers, researchers, and educators submit features, fixes, integrations (e.g., MolViewSpec compatibility, enhanced Jupyter support, AI-assisted scene building, VR/immersive modes), and content via pull requests to the main branch.

Transition to self-sustaining OSE: Forking NCBI's iCn3D under VIZomics governance to foster distributed development, clear roadmap, and broader adoption in mechanistic biology.

Enables virtuous cycle: Promotes open data ↔ open source ↔ open science through reusable, shareable molecular scenes that support reproducible research, interactive education, ensemble analysis (AlphaFold, EM, MD trajectories), and innovative publishing.
